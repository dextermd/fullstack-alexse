ФАСОВОЧНЫЙ ДОЗАТОР

Дозатор весовой для фасовки сыпучих компонентов используется для расфасовки в открытые мешки и пакеты массой от 20 до 50 кг зерна, комбикорма, отрубей, гранул, удобрений, семян, пеллет, сахара, муки и других с ними схожих материалов.
Фасовочный дозатор крайне прост и удобен в эксплуатации, мешок с материалом взвешивается в процессе наполнения, при достижении заданной массы наполнение мешка прекращается автоматически. После заполнения, мешок заменяется на новый.
Весовой терминал отображает текущий вес набранной дозы, количество взвешенных мешков и их суммарную массу, имеет удобное управление и широкие возможности по встройке в технологические линии.
Данные о взвешиваниях сохраняются в энергонезависимой памяти и не могут быть изменены оператором вручную.

ПОЛУАВТОМАТИЧЕСКИЙ ФАСОВОЧНЫЙ ДОЗАТОР

  • Работает без использования сжатого воздуха
  • Ручной зажим мешка.
  • При необходимости комплектуется ленточным транспортером для перемещения мешков в горизонтальном направлении. Существенно облегчает работу оператора
  • При необходимости комплектуется зашивочной машинкой для зашивки мешка после наполнения – устанавливается на специальной стойке, управляется ножной педалью
  • Для некоторых продуктов в состав дозатора могут быть включены: встряхиватель - для уплотнения продукта в мешке
  • Для фасовки пищевых продуктов, части, контактирующие с материалом, выполняются из нержавеющей стали

ПНЕВМАТИЧЕСКИЙ ФАСОВОЧНЫЙ ДОЗАТОР

  • Ручной или пневматический зажим мешка.
  • При необходимости комплектуется ленточным транспортером для перемещения мешков в горизонтальном направлении. Существенно облегчает работу оператора
  • Удобное открытие заслонки педалью, для начала наполнения мешка.
  • При необходимости комплектуется зашивочной машинкой для зашивки мешка после наполнения – устанавливается на специальной стойке, управляется ножной педалью
  • Для некоторых продуктов в состав дозатора могут быть включены: встряхиватель - для уплотнения продукта в мешке, вибратор - для предотвращения • Для некоторых продуктов в состав дозатора могут быть включены: встряхиватель - для уплотнения продукта в мешке, вибратор - для предотвращения
  • Для фасовки пищевых продуктов, части, контактирующие с материалом, выполняются из нержавеющей стали.
Бункерные весы Alex S&E типа – BSN, соответствуют стандарту RGML14:2007 «Весы неавтоматического взвешивания». Сертифицированы и внесены в Государственный реестр средств измерений Республики Молдова.
