Кухня

Модуль для повара

В стандартной поставке на кухню или в каждый цех кухонного блока устанавливается принтер для печати встречек. Установка компьютера с модулем для повара существенно расширяет возможности системы. Записи заказов, которые распечатываются на принтере для встречек, автоматически появляются на экране компьютера. Повар по каждой записи может отметить факт принятия заказа в работу, при необходимости создать требование на перемещение ингредиентов со склада. Отметка факта выполнения заказа отправляет на POS-терминал сообщение о готовности заказа. Если официант, отправивший заказ, оснащен смартфоном, он получит сообщение об этом на телефон. Дополнительным преимуществом является возможность контролировать скорость выполнения заказов.
