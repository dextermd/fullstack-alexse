1. Доставка

    Модуль для call-центра

    Сотрудник call-центра, принимая звонок, автоматически идентифицирует постоянного клиента по номеру телефона. Есть возможность подставить один из введенных ранее адресов доставки клиента или ввести новый адрес. Программа автоматически рассчитает расстояние от производства до адреса доставки и может сформировать стоимость доставки. Список блюд заказа уходит на производство.

  2. Доставка

    Модуль выдачи заказа

    Укомплектованный заказ может быть идентифицирован этикеткой со штрих-кодом для предотвращения путаницы при отправке клиенту, распечатывается счет для оплаты.

  1. Доставка

    Модуль для производства

    Производственный модуль позволяет автоматизировать учет всех производственных процессов, контролировать остатки ингредиентов и полуфабрикатов, производить списание по калькуляционным картам на каждом этапе производства. Готовая продукция поступает на точку выдачи заказа.

  2. Доставка

    Мобильный модуль агента-водителя

    Водитель сканирует этикетку на укомплектованном заказе, тем самым отмечая в мобильном модуле факт получения заказа. Модуль прокладывает путь на карте. Факт оплаты заказа так же отмечается в программе и передается через 3/4g в центральную базу.
