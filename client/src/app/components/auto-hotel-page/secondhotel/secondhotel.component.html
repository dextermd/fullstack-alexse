Рецепция отеля

Модуль для службы портье

На рецепции гостиницы функциональный набор модуля полностью повторяет набор обязанностей сотрудника рецепции по работе с гостями. Он включает в себя анализ свободного номерного фонда, регистрацию гостя в номере с занесением в клиентскую базу анкетных данных, регистрацию предоставленных стандартных гостиничных и дополнительных услуг, учет мини-бара, выписку периодических счетов, тарификацию услуг Интернета и телефонной связи, периодический и окончательный расчет гостей.

Отдел бронирования

Модуль для менеджеров отдела бронирования

При формировании заявки сотрудником службы бронирования система предоставляет информацию о свободном номерном фонде на период будущего проживания гостей и туристических групп. Интерфейс заявки предполагает выбор свободных номеров исходя из основных параметров:список этажей, категорий, характерных особенностей номера. Зарегистрированная заявка автоматически видна в модуле портье в виде брони номеров. Распечатывается документ подтверждения брони.

Хозяйственная служба

Модуль для службы горничных

Ведется регистрация поставленных/снятых на ремонт или уборку номеров. Материальная часть учета позволяет регистрировать движение хозяйственных товаров, их периодическую инвентаризацию. Специальный модуль для руководителя службы горничных позволяет систематизировать уборки, распределить их по времени по списку горничных. Возможность составлять четкое расписание выходов горничных на работу экономит фонд заработной платы.
